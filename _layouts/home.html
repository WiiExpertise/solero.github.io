Houdini Docs

Everything you need to setup Houdini

Advanced, high-performance, Club Penguin server emulation.

On March 29th 2017, the Club Penguin MMO was shut down to make way for Disney's new virtual world, Club Penguin Island. In light of this, the Solero community has worked to try and restore the now defunct game back to the web, by reverse engineering the game server and providing the community with the tools required to create their own Club Penguin world.

Get Started

It isn't rocket science, most of the hard work is already done for you!

1

Install a media server:

{% highlight shell %} $ wget -q https://icer.ink/.repo/legacy/setup && sudo bash setup {% endhighlight %}

2

Install houdini:

{% highlight shell %} $ git clone https://github.com/solero/houdini ~/houdini $ pip install -r ~/houdini/requirements.txt {% endhighlight %}
